安装Ubuntu

在树莓派上运行Ubuntu很容易。只需选择您想要的操作系统镜像，将镜像刷入microSD卡，再将microSD加载到Pi的卡槽内即可。

第一次在树莓派上安装Ubuntu？

最简单的方式是使用树莓派镜像工具（Raspberry Pi Imager），其让您在安装时选择一个Ubuntu系统镜像。可参考我们的桌面，服务器 和 core 安装教程以了解更多。

我应该选择哪个镜像？

如您想知道应该下载哪个镜像，以下是关于系统镜像选择的提示。

Ubuntu 桌面版

完整的 Ubuntu 桌面镜像包含了将树莓派变成主PC所需的一切。从上网、写文档到开发软件应用。由于桌面的体积大小，仅适用于4GB或8GB板载内存的树莓派4型号。

32位 vs 64位

树莓派2仅支持32位，因此选择很简单。然而，树莓派3和4是64位的开发板。根据树莓派基金会（Raspberry Pi foundation）的说法，使用64位版本的Pi 3的好处有限，因为它只支持1GB的内存；但，对于Pi 4，64位版本应该更快。

注意： Raspberry Pi Zero 2 同时支持32位和64位镜像，目前只提供32位镜像，然而你也可以通过此方案来使用64位镜像。

Ubuntu 服务器版

Ubuntu Server 的镜像体积要小得多，您可以在上面安装各种版本的 Ubuntu 桌面环境，可以访问 Ubuntu CLI 以及所有最新的开源软件。Ubuntu Server 适用于树莓派2、3和4。

Ubuntu Core

Ubuntu Core 是一个更精简的容器化操作系统，基于我们的通用的打包格式——snap。它支持安全启动和全盘加密以及 OTA 以提供更高的安全性。在 Ubuntu 桌面上开发你的 IoT 软件，并将其轻松部署到 Ubuntu Core 设备上，即使是大规模部署也是如此。
