Ubuntu Core

Ubuntu Core 是专为物联网和边缘计算而优化的 Ubuntu 版本，具备强大的安全模型。Core 广泛应用于物联网产业的大小型企业，已有大量的群组设备部署在实际应用场景中。

Ubuntu Core 与 classic Ubuntu 采用相同的内核、库和系统软件，进而实现从开发到生产的平滑过渡，并默认包含 Ubuntu Pro 。
