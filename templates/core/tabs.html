Ubuntu 是您的开发环境。Ubuntu Core 是您的部署操作系统。

无论您使用 C++、Rust、Python 还是其他语言来开发应用程序，无论采用何种技术栈，都可在 Ubuntu Core 上实现无缝打包与部署。

Snapcraft 是用于将软件打包为 snap 的命令行工具。Snaps 是为部署嵌入式应用程序而设计的容器化包，与其他技术相比，性能更佳。Ubuntu Core 架构基于 snaps 构建，每个组件都受益于其工具。

Canonical 的 Snap Store 让开发人员能够托管，并管理其设备的软件更新。这个安全可靠的应用商店包括 OTA 更新、delta 更新等功能。

企业更可选择为自己的设备创建专属的私有应用商店，实现对设备群的软件管理。
