Ubuntu Kylin / 优麒麟

Ubuntu是世界最受欢迎操作系统之一，目前拥有超过2500万的桌面用户排名第三，而它在服务器和云计算上的应用更为广泛。它是一款完全开源的由国际软件商家Canonical作为坚实后盾的操作系统。因为它对不同硬件的支持和易用，以及它至核心的安全机制，目前在全世界有众多商业，政府及教育机构已经采用Ubuntu操作系统。

Ubuntu Kylin （中文又被称为优麒麟）是基于Ubuntu的一款官方衍生版。 它是一款专门为中国市场打造的免费操作系统，而且它已经被录入中国政府采购条例名单中。它包括Ubuntu用户期待的各种功能，并配有必备的中文软件及程序。与竞争对手不同，优麒麟的用户界面专门为中国用户设计，并且支持触摸屏以及HiDPI高清显示屏，在各种不同硬件上都可以完美运行。

优麒麟背后的团队

优麒麟是由 Canonical，工业和信息化部软件与集成电路促进中心（CSIP）以及国防科学技术大学（NUDT）携手打造。

目前，开发团队正在与开源社区合作打造Ubuntu的核心功能，以更好地适应中国用户。 除了Ubuntu软件中心推出的本土化版本以外，优麒麟还带最常用的中文程序。

{# ================ 6. Sogou Input Method ================ #}

搜狗输入法

优麒麟为中文输入精心打造的搜狗输入法，允许字词语句自动补全。更重要的是，它具有互联网搜索的云输入功能，不论是浏览网页或创建新文件，都能使你的优麒麟电脑使用起来更方便更快捷。最新发布的优麒麟15.10版本新增了搜狗输入法的双拼输入。

{# ================ 7. Youker Assistant ================ #}

优客助手

优客助手是一个协助Windows用户在最短的时间内以最便捷的方式切换到Ubuntu操作系统的简单有效的工具。这个内建的工具有助于日常的系统管理工作，比如硬件检测和界面个性化处理等。有了优客助手，您就再也不必担心如何使用优麒麟了。最新版本优化程序架构，更加节省系统资源，增加文件粉碎机功能、UKID登陆、西班牙语言支持和Cinnamon桌面环境的美化配置，并完善进程管理器。

{# ================ 8. Smart Scopes ================ #}

智能搜索

用户在优麒麟Dash中可以发现专用的本地化智能搜索，它将使在线内容的搜索变得更快捷更有针对性。通过智能搜索，用户可以直接搜到搜狗在线的图片内容， 优酷的视频内容，以及来自百度的音乐内容；这一切都会和本地内容搜索的结果同时呈现。智能搜索让你在桌面上的内容搜索更聪明广泛。
