Ubuntu Kylin 14.04 LTS 正式发布!

  • Ubuntu Kylin 14.04 LTS 操作系统适用于政府、企业和普通消费者，并提供 长达五年的支持和维护；
  • 是替换 Windows 系统的一种可行、更具成本效益且安全可靠的方案；
  • 系统的易用性、稳定性和丰富功能已得到大量消费者的认可。

2014 年 4 月 22 日，由工业和信息化部软件与集成电路促进中心(CSIP)携手国防 科技大学 (NUDT) 和 Canonical 公司合作成立的 CCN 开源软件创新联合实验室 在北京召开新产品发布会，宣布 Ubuntu Kylin 14.04 LTS 版本正式发布并确定中 文名为“优麒麟”。此次发布的长期支持(LTS) 版拥有长达五年的支持和维护，使 其成为适合个人应用、大型企业和政府机构办公的最稳定、可靠且更具成本效益 的桌面操作系统，尤其有助于用户从服务已经到期的 Windows XP 系统切换到本 系统。中国开源软件推进联盟名誉主席陆首群，中国工程院院士倪光南，金山集 团高级副总裁葛珂，搜狗公司副总裁杨洪涛，浪潮软件公司企业发展部总经理方 亚东，君盾集团董事长李必录，华文字库董事长黄克俭，LUPA 主席张建华，CSDN 董事长蒋涛，Deepin 开源操作系统创始人刘文欢，中标软件相关负责人等嘉宾 参加了此次发布会，一起见证了这一历史时刻。与会嘉宾共同探讨了如何打造更 加优秀的 Ubuntu Kylin 操作系统，并将其推广到更多政府和企业应用中等多个议 题。

CCN 开源软件创新联合实验室的成立旨在打造专门满足中国用户需求的操 作系统。为了实现该目标，Ubuntu Kylin 操作系统基于全球领先的 Ubuntu 开源 系统，并从 13.04 版本开始，得到 Ubuntu 的官方认可和支持。Ubuntu 已经创立 10 年，被全球超过 2000 万用户和商家使用。Ubuntu Kylin 14.04 LTS 是 Ubuntu Kylin 操作系统的第三个正式发行版，也是首个长期支持 (LTS) 版，提供最新的 功能和工具，以确保直观而舒适的用户体验。

在该版本中，Linux 内核升级到了 3.13，Unity 升级到了 7.2。专为 Linux 普通 用户打造了图形化的系统级管理与配置工具——优客助手，该软件具备强大的系 统信息展示、一键垃圾清理、系统定制美化等功能，让 Ubuntu Kylin 更加流畅、 易用而富有个性！为了实现高效的中文输入，Ubuntu Kylin 团队与搜狗公司联合 开发了搜狗输入法 Linux 版本，该输入法具有智能预测搜索、自动匹配功能、多 样化皮肤支持等功能。搜狗输入法与金山 WPS 办公套件相结合，使 Ubuntu Kylin 14.04 LTS 能够更加快捷而轻松地进行文档编辑与日常办公。 新版本还包含金 山快盘客户端，提供了最高 2TB 的个人云存储空间，可以通过互联网快速访问 任何兼容金山快盘的设备上的文件和内容。轻量级的 Ubuntu Kylin 软件中心同时 支持 Ubuntu 和 Ubuntu Kylin 软件仓库。用户可以轻松享用 Ubuntu Kylin 团队 与金山办公、金山云、搜狗等公司合作推出的中文流行应用软件！

Ubuntu Kylin 14.04 LTS版本还针对最新的台式计算机和笔记本电脑以及多 种触控设备，如触控板和触摸屏、高像素密度（DPI）屏幕键盘和鼠标等，进行 了优化设计。这意味着用户可以体验专为手机和平板电脑设计的触控类应用程 序，如触控设备上的新闻阅读器、图库和相机。

国防科技大学计算机学院院长廖湘科教授表示，“操作系统是信息系统的基 石，实现操作系统的国产化，一直是党和国家领导人高度关注的问题。基于开源 软件研制自主的基础软件是我国基础软件自主可控发展的必由之路。优麒麟项目 采用中国主导、国际合作的开源社区模式，已经取得初步成功，在系统定制、应 用开发、生态构建、软件中心、社区建设等方面开展了持续而有效的工作。后续 版本中，我们会继续完善现有成果，并在 SDK、UI、桌面环境等方面开展自主 研发和国际合作，目前我们通过与 WPS、金山快盘、搜狗输入法的深度合作， 探索了一条逐步打造我国开源操作系统生态环境的可行路线。后面我们还会与国 内外其它开源社区合作，共同扛起中国开源操作系统的大旗，让国人有更好操作 系统桌面体验！”

工业和信息化部软件与集成电路促进中心主任、中国开源软件推进联盟主席 邱善勤博士指出，“一年多来，我们与 Canonical、国防科大成立的 CCN 联合实 验室取得了可喜的成绩，一个由中国人主导的、与国际国内社区融合发展、合作 创新机制下的开源操作系统优麒麟已连续发布了 3 个社区版本，今天发布的第三 个版本 14.04 LTS 长期支持版是个重要的里程碑，这表明一是优麒麟开源社区版 逐渐走向成熟；二是我们所探索的以国内外开源社区合作创新模式推动我国基础 软件发展的方式，正走向正确的道路上。接下来，我们将继续探索这种社区合作 创新机制，“CCN 开源软件创新联合实验室”也将继续遵循 “开放、共享”原则， 借鉴学习世界领先的开源社区运营经验，凝聚社区力量。欢迎更多的业界力量加 入，优势互补、协同创新，为中文用户提供更加舒适的操作体验，为开源软件的 发展贡献力量。”

“仅过去的一年，Ubuntu Kylin 的官网下载量已经超过 300 万次，并且已经在 中国得到了大量用户和大型组织机构的认可与试用。许多企业将 Ubuntu Kylin 视为一种更安全、更具成本效益、易于管理且安全可行的替代操作系统，适合取 代服务到期的微软 XP 系统。使用 14.04 LTS 版本的企事业单位将有机会从 CCN 联合实验室获得最新的软件以及持续的支持和维护，”Canonical 首席执行官 Jane Silber 评论说。

Canonical 致力于面向全球用户提供针对 Ubuntu 的管理和支持服务，并与戴 尔、惠普和联想等 OEM 合作，每年将数百万台预装 Ubuntu 的设备发送至世界 各地。目前，戴尔在中国拥有 1000 多个直接向客户销售 Ubuntu 系统设备的零售 门店。未来 Ubuntu Kylin 14.04 LTS 版本也将在合作伙伴的硬件上获得认证与应 用。

14.04 LTS 版本的其他主要特色包括：采用百度搜索引擎及搜狗门户网站作 为起始页的火狐/谷歌浏览器，优客助手，金山 WPS 办公套件以及 Dash 中文搜 索等，并提供专业的中文化体验。若需了解完整的特色列表，请访问： http://www.ubuntukylin.com/applications/

关于 NUDT

国防科技大学是国防部和教育部双重领导下的国家重点综合性大学，列入国家 “985 工程”和“211 工程”的重点建设。学校的前身是 1953 年创建于黑龙江省哈尔 滨市的军事工程学院，简称“哈军工”。该校研制的“天河”系列超级计算机在过 去四年中三次问鼎世界超级计算机“Top 500”榜首。

关于 CSIP

工业和信息化部软件与集成电路促进中心 CSIP 是工业和信息化部的直属事业单 位，主要负责国家软件与集成电路公共服务平台的建设工作，是国内软件产业和 集成电路产业发展的积极推动者。CSIP 秉承“促进产业发展、助力企业创新”的 宗旨，致力于为我国软件与集成电路产业和企业发展提供公共、中立、开放的服 务。

关于 Canonical 和 Ubuntu

Canonical 是 Ubuntu 的创始公司，也是企业 Ubuntu 部署的领先服务商。凭借国 际开发团队的力量，技术支持人员和工程中心，Canonical 具有得天独厚的优势， 帮助合作伙伴和客户更好的运用 Ubuntu 。Ubuntu 是一个面向客户端，服务器和 云计算的开源平台。这是最广泛使用的 Linux 系统（从排名前 1000 的网站流量 截得），也是 OpenStack 部署的参考平台，是公共云以及戴尔、惠普、联想等电 脑上最流行的客户机操作系统。
