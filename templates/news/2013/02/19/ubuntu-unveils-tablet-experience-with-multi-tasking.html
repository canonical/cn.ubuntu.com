Ubuntu多任务平板电脑体验

  • 独特的“多窗”(side stage) 多任务模式可将手机和平板电 脑应用程序显示在同一平板电脑屏幕上。
  • 使用全磁盘加密技术确保企业平板电脑信息安全，为用户 帐户和标准管理工具（涵盖 Ubuntu 服务器、个人电脑和触 控式产品）提供多重安全保护。
  • 完美融合四种主板尺寸：将手机接入合适的屏幕/键盘/遥 控器即可使用平板电脑、电视和个人电脑界面。

2013 年 2 月 19 日,伦敦: Canonical 今日发布了 Ubuntu 平板电脑界面—展示更上层楼的手机、平板电脑、个人电脑和电 视全方位个人计算机综合体验。

"我们的平板电脑支持多任务处理功能,不仅时尚优雅,而且安 全可靠”,Ubuntu 和 Canonical 创始人 Mark Shuttleworth 如是说。“我们现在推出的界面适用于各种屏幕尺寸,只要您接 入手机即可享受平板电脑、个人电脑和电视体验。这是 Ubuntu 的特色,也是个人计算机的发展趋势。"

时尚界的朋友说 Ubuntu 的手机和平板电脑界面是他们见过 的同类触控式产品中最漂亮的”，Canonical 设计团队领军 Ivo Weevers 说。“我们的创作灵感源自实现时尚和实用性双 重目标，我们的开发商旨在为朋友、家人和业内人士提供最佳用 户体验。

新平板电脑不仅外观更加时尚美观，而且在设计和工程技术层 面有新突破，具有以下功能：

  • 真正多任务处理功能：Ubuntu 与众不同的一点是允许在屏幕 上同时运行手机应用程序和平板电脑应用程序。Ubuntu“多 窗”(side stage) 功能不仅能实现高效多任务操作，还能提高手 机应用程序在平板电脑上的实用性。
  • 多用户信息安全：对同一平板电脑上多个帐户的个人信息充分 加密；此外，还采用 Ubuntu 安全模型，该模型安全可靠，被广 泛应用于银行、政府和多类敏感环境中。是商用和家用平板电脑 的理想之选。
  • 声控平视显示器 (HUD) 效能：Ubuntu 独特的平视显示器能 在触控设备上简单而快速地处理复杂任务，能转换触控界面使 之运行诸多应用，为您带来完美个人电脑体验。
  • 边幻效果更简洁美观地呈现应用程序集：可在屏幕边缘切换 应用程序导航，进行设置和实施控制。这便于操作管理，能显示 更多内容，无需过多硬件设备。无须物理按钮或软按钮。可为用 户提供纯粹触控体验。
  • 内容聚焦：可自定义主屏幕，能整洁美观地显示媒体资料，可 以搜索数百个数据来源。非常适合那些想凸显自己内容的运营 商和内容所有者，同时可查看总体目录。
  • 完美融合：平板电脑界面使用与手机、个人电脑和电视界面完 全一样的操作系统和代码，可真正实现各设备完美合。Ubuntu 的独特设计旨在完美融合各种机型。

Ubuntu 平板电脑界面支持的屏幕尺寸为 6 英寸(15.2厘米)至 20 英寸（50.8厘米），分辨率为 100 至 450像素/英寸(PPI)。 “Ubuntu 平板电脑非常匹配 Ubuntu 手机和个人电脑” ，Ubuntu 平板电脑首席设计师 Oren Horev 介绍说。“我们不 仅以独特的方式集成手机应用程序，而且还通过融合设备非常 顺利地从平板电脑转换到个人电脑。

Ubuntu 平板电脑使用高端芯片，接入键盘后即可提供完整个 人电脑体验，还可基于微软、思杰、VMWare 和 Wyse 标准协 议远程访问 Windows 应用程序。 Canonical 企业台式机和 瘦客户机产品领头人 Stephane Verdy 说“Ubuntu 平板电脑 是安全的瘦客户机，可使用 Ubuntu 服务器或台式机的工具对 之进行管理。 我们非常高兴能在企业市场为合作伙伴提供触摸 式移动瘦客户机支持”。

即使不优化芯片组性能，Ubuntu 也能在入门级硬件上完美运 行。“我们与 ARM 合作了四年，Ubuntu 已经准备好进军移动 领域”，Canonical Ubuntu 工程副总裁 Rick Spencer 说。“ 我们拥有大量的开发人员，他们每天使用 Ubuntu，其中许多开 发人员主要基于触控设备进行开发，贡献颇丰。

对芯片厂商而言，Ubuntu 能与任何面向 Linux 的板级支持包 (BSP) 兼容。这意味着 Ubuntu 很容易在当前使用 Android 的 大多数芯片组上运行。Ubuntu 和 Android 是 Linaro 成员启 用的两个平台。

触控版 Ubuntu 开发者预览 (Touch Developer Preview) 将 于 2013 年 2 月 21 日发布，配备 Nexus 7 和 Nexus 10 平板设 备及智能手机（如 Nexus 4 和 Galaxy Nexus）安装说明。可登 录 developer.ubuntu.com 查看安装映像和源代码。

预览版 SDK(Preview SDK) 目前支持手机应用程序开发，即将 更新以支持平板电脑应用程序。Ubuntu 与众不同的一点是，开 发人员可以创建同时适用于手机、平板电脑、个人电脑和电视的 单一应用程序，这是因为所有设备使用同一系统且全部服务可 在所有机型上运行。

关于 Canonical 和 Ubuntu

Canonical 是 Ubuntu 的技术支持公司，是为部署 Ubuntu 的 企业提供服务的领先供应商。Canonical 的开发团队、技术支 持人员和工程中心遍布全球，拥有独一无二的优势，能帮助其合 作伙伴和客户最充分地利用 Ubuntu。该公司还负责运作 Ubuntu One，为消费者提供跨平台个人云服务。Canonical 是一家私人控股公司。

Ubuntu 是一个免费的开源代码平台，供客户端、服务器和云计 算使用。它是流量排名前 1000 名网站上最广泛使用的 Linux 系 统，是 OpenStack 部署的参考平台，也是公共云最流行的客户 机操作系统，适用于戴尔、联想、惠普等品牌的个人电脑。自 2004 年推出以来，其已成为开放桌面和横向扩展计算的首选， 《财富》500 强公司、硬件制造商、内容提供商、软件开发商和 消费者都对其青睐有加。
